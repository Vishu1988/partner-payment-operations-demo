# Partner Payment Operations Demo

Lightweight onboarding and payment operations dashboard prototype built using Bolt.

## Overview
This project simulates partner onboarding workflows, onboarding visibility, operational review ownership, issue prioritization, and integration health monitoring for payment operations teams.

## Features
- Partner onboarding state tracking
- Integration health monitoring
- Review ownership visibility
- Issue severity prioritization
- AI-assisted onboarding summaries
- Workflow visibility and filtering

## Purpose
Built as a rapid prototype to explore how onboarding friction, operational visibility, and workflow coordination can be simplified through lightweight tooling and AI-assisted workflows.

## Tech Stack
- Bolt
- React
- TypeScript
- Tailwind CSS
